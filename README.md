## iOS  
### nPlayer  
1. 在App Store里安装nPlayer（中国大陆区可用）  
<https://apps.apple.com/cn/app/nplayer-lite/id1078835991>  
2. 在nPlayer里配置本站WebDAV源  
- WebDAV配置内容  
  - 标题：随意填写  
  - 主机：cafe.ycmt.top  
  - 用户：anime  
  - 密码：anime  
  - 密码锁：不开启  
  - 端口：80  
  - 路径：/dav  
  - HTTPS：不开启
